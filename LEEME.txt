Landing page HGR editada con logo oficial integrado.

Para publicar en GitHub Pages:
1. Descomprime este ZIP.
2. Sube TODO el contenido al repositorio, reemplazando archivos anteriores.
3. Verifica que index.html quede en la raíz del repositorio.
4. GitHub Pages debe estar en: main / (root).

Incluye:
- Logo en header, portada, favicon y footer.
- Botones de WhatsApp, ubicación y PDF.
- Cuenta regresiva al evento.
